Este sistema é uma aplicação web em Flask que permite gerenciar uma livraria digital.

Ele possibilita o cadastro, consulta, atualização, exclusão e exportação de livros, além de importar dados de vendas e clientes.

Também realiza backups automáticos do banco de dados SQlite, gera relatórios em HTML e PDF, e oferece rotas de API RESTful para integração externa.
